# Goldenblue

Basically Fedora Silverblue, except my own golden image version of it... get it?

With thanks to [Seraphim Strub](https://dev.rievo.net/sst/uphy) for the original GitLab CI friendly version of this code, which is in turn based on much of the [uBlue](https://ublue.it/) project's work.

### Notes
#### /etc/vconsole.conf
Sets the console font to `latarcyrheb-sun32` which is a large size font. Seems to be a good choice for high DPI displays so I left this intact.
